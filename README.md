# 🎵 Spotify Clone

A responsive front-end clone of the Spotify music streaming platform, built using **HTML**, **CSS**, and **JavaScript**. This project aims to mimic the look and feel of the original Spotify UI, with custom styling, interactivity, and a clean layout.

## 🚀 Features

- 🎧 Custom audio player with play/pause functionality
- 🖼️ Stylish UI inspired by Spotify's web interface
- 📱 Responsive design for desktop and mobile
- 🔁 Playlists and song list simulation
- 🔊 Volume and progress control (JavaScript-based)

## 🛠️ Tech Stack

- **HTML5** – structure and content
- **CSS3** – styling and layout (Flexbox & Grid)
- **JavaScript** – interactivity and audio control
